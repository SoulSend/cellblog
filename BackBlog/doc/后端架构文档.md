# 后端架构文档

## 架构概述

BackBlog 采用经典的三层架构设计，基于 Spring Boot 框架构建，使用 MyBatis Plus 作为 ORM 框架，MySQL 作为数据库。

## 技术栈

### 核心框架
- **Spring Boot 3.3.4**: 主框架
- **MyBatis Plus 3.5.7**: ORM 框架
- **MySQL 8.0**: 数据库

### 安全认证
- **JWT**: JSON Web Token 认证
- **MD5**: 密码加密

### 文件存储
- **七牛云**: 对象存储服务

### 工具库
- **Lombok**: 代码简化
- **FastJSON**: JSON 处理
- **Commons Codec**: MD5 加密
- **Joda Time**: 时间处理
